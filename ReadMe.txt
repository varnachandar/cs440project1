INTRODUCTION
-----------
This project uses pythons Tkinter libary inorder to visulize A* and theta* search algorithms.

How to run
-----------
1.First command input the name of your grid file, with format "grid.txt"
2.Input weather you want to run theta* by entering Theta or A* by entering A
3. Once the grid has been developed you can expand any of the buttons to reveal the G, H and F values.

Disclaimers 
-----------
This project was built and tested on desktops therefore if ran on smaller screens sizing issues may arise.
